Lesson 1: SEO Fundamentals I

The Search Process

When a user performs a search on Google or any other search engine, a complex process is triggered to provide the most relevant and useful results. This process involves several key stages: crawling, indexing, and ranking.

Crawling

Search engines use automated programs called "bots" or "spiders" to browse the web and discover new or updated content. These bots follow links from one page to another, gathering information about each page they encounter.

Indexing and Organisation

Once the content is crawled, it is stored in an index—a large database of information that the search engine uses to respond to user queries. During indexing, search engines analyse the content of pages, their structure, their relevance to certain keywords, and other factors important for ranking.

Ranking

When a user makes a search, the search engine consults its index and returns results according to various factors, such as the relevance of the content to the query, page authority, content quality, and user experience. The search engine can learn in real-time from user behaviour and adjust the results accordingly.


Understanding how this process works is fundamental for optimising a website. Knowing how people search and how search engines determine the most relevant content allows us to adapt our SEO strategies to improve ranking.
